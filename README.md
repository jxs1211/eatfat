### project structure
```sh
/
├── server/             # Go backend server
│   ├── cmd/           # Command line applications
│   ├── internal/      # Private application code
│   ├── pkg/           # Public libraries
│   ├── go.mod         # Go module file
│   └── Makefile       # Build automation
├── client/            # Godot client
│   └── scenes/        # Godot scenes
└── README.md          # Project documentation
```
